# Font-Installer-Ruby
Font InstallerのRuby版です。

### 実行方法
管理者権限で実行してください。

sudoなどでもいいです。

#### Windows
```cmd
ruby install.rb [フォント名]
```

#### 他のOS
```sh
ruby ./install.rb [フォント名]
```

### 依存関係

#### ソフトウェアなど
```
Ruby
```

#### Rubyライブラリ
()内はgemでのパッケージ名です。
```
FileUtils(fileutils)
Rubyzip(rubyzip)
```
