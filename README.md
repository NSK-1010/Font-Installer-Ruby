# Font-Installer-Ruby
The Ruby version of the Font Installer.

## How to run.
Run it with administrative privileges.

You can also use sudo.

### Windows.
```cmd
ruby install.rb [font name]
```

### Other OSes.
```sh
ruby ./install.rb [font name]
````

## Dependency.

### Software, etc.
```
Ruby
```

[Ruby's website is here](https://www.ruby-lang.org/)
